# 帮派树挂机与建筑解锁系统设计

## 目标

在城市 Demo 中加入 1–50 级帮派树。玩家通过在线与离线挂机持续获得帮派声望并提升等级；职位随等级变化，建筑按指定顺序解锁。玩家初始只拥有修车厂，未解锁建筑在城市中显示为锁定地块，不能升级。

## 明确映射

用户给出 6 座建筑和 7 个职位。为同时满足建筑顺序与“50 级对应主席”，采用以下映射：

| 起始等级 | 职位 | 建筑解锁 |
|---:|---|---|
| 1 | Prospect（见习） | 修车厂 |
| 8 | Full Patch（正式成员） | 废车回收厂 |
| 16 | Wrench（技术骨干） | 商业街 |
| 24 | Bar Liaison（酒吧联络人） | 金属加工厂 |
| 32 | Road Captain（路线队长） | 加油站 |
| 40 | V. PRESIDENT（Vice President，副主席） | Clubhouse |
| 50 | PRESIDENT（主席） | 无新建筑，作为帮派树满级终点 |

等级区间：

- 1–7：Prospect
- 8–15：Full Patch
- 16–23：Wrench
- 24–31：Bar Liaison
- 32–39：Road Captain
- 40–49：V. PRESIDENT
- 50：PRESIDENT

## 城市建筑调整

现有六座城市建筑调整为：

1. `repair-shop`：修车厂
2. `recycling-yard`：废车回收厂
3. `commercial-street`：商业街
4. `metalworking-plant`：金属加工厂
5. `gas-station`：加油站
6. `clubhouse`：Clubhouse

现有物流中心替换为金属加工厂；商业区改为商业街；废车回收站改为废车回收厂。建筑中心位置与城市主要构图保持不变。

金属加工厂三级外观：

- 1 级：低矮加工车间、基础熔炉与材料堆。
- 2 级：扩建冲压车间、增加烟囱与吊装架。
- 3 级：大型金属加工主楼、双炉体、重型天车与高烟囱。

商业街沿用原商业建筑体块，但名称、说明和识别标签改为商业街语义。

## 帮派声望与挂机公式

首版 Demo 使用固定、可读的线性公式：

- 每级需要 30 声望。
- 在线挂机每秒获得 5 声望。
- 等级公式：`min(50, floor(totalReputation / 30) + 1)`。
- 50 级所需累计声望：1470。
- 从 1 级到 50 级约需 294 秒，适合 Demo 内观察完整解锁流程。
- 50 级后停止增长，声望封顶 1470。

离线挂机：

- 使用本机 `localStorage` 保存累计声望与最后结算时间。
- 页面加载、标签页重新可见和每秒 tick 时结算经过时间。
- 最多累计 8 小时离线收益。
- 系统时间倒退或无效时间不产生收益。
- 达到 50 级后不继续累计。
- 本功能是本地 Demo 存档，不涉及账号或后端。

## 帮派树界面

城市 HUD 增加：

- 当前等级，例如 `Lv. 12`。
- 当前职位中英文。
- 本级声望进度条。
- 当前挂机速率 `+5 声望/秒`。
- “帮派树”按钮。

点击按钮打开帮派树面板：

- 使用覆盖全屏的深色半透明 modal，内部为可滚动纵向树。
- 展示 1–50 共 50 个等级节点，不合并等级。
- 节点状态分为：已完成、当前、未解锁。
- 职位阈值 1/8/16/24/32/40/50 使用更大的里程碑节点。
- 建筑解锁节点展示建筑名称与“已解锁/待解锁”。
- 当前等级自动具有明显高亮。
- 顶部显示当前职位、总声望和下一职位目标。
- 提供关闭按钮；Escape 可关闭。
- modal 打开时阻止点击穿透到 3D 场景。
- 小屏幕改为全屏面板。

## 锁定建筑表现

- 玩家初始仅拥有修车厂。
- 其他五座建筑仍保留在城市布局中，但显示为低矮深色施工地基或半透明锁定轮廓。
- 锁定地块可悬停和点击，以便了解解锁条件。
- 选中锁定地块时，建筑面板显示：
  - 建筑名称
  - “尚未解锁”
  - 所需帮派等级与职位
  - 当前等级进度
- 锁定建筑不显示建筑等级升级按钮。
- 达到解锁等级后，锁定地块立即替换为该建筑 1 级模型。
- 已解锁建筑保留原有 1–3 级升级系统。
- 帮派等级下降不属于本系统，因此不存在重新锁定。

## 技术架构

### 纯数据与规则

- `src/game/gangProgression.ts`
  - 1–50 等级常量
  - 七个职位定义
  - 六座建筑解锁定义
  - 声望/等级换算
  - 当前职位与下一职位
  - 建筑是否解锁
  - 在线/离线收益计算

### 持久状态

- `src/store/useGangStore.ts`
  - `totalReputation`
  - `lastUpdatedAt`
  - `syncIdleProgress(now)`
  - `reset(now)`
  - Zustand persist，存储键 `gang-progression-v1`

城市建筑等级继续由 `useCityStore` 管理；建筑所有权只从帮派等级派生，不复制到城市 store。

### 运行控制

- `src/game/GangIdleController.tsx`
  - 页面挂载时结算离线收益
  - 每秒结算在线收益
  - `visibilitychange` 回到前台时补算
  - 卸载时清理 timer 和事件

### UI

- `CityHud`：显示等级、职位、进度与帮派树入口。
- `GangTreePanel`：50 级树、职位和建筑解锁里程碑。
- `BuildingPanel`：根据解锁状态显示锁定信息或建筑升级。

### 场景

- `InteractiveBuilding`：根据当前帮派等级渲染完整建筑或 `LockedBuildingPlot`。
- 锁定地块仍复用完整 footprint 命中盒。

## 数据流

1. `GangIdleController` 调用 `syncIdleProgress(Date.now())`。
2. store 根据经过秒数增加声望并持久化。
3. `getGangLevel(totalReputation)` 派生 1–50 级。
4. `getGangRole(level)` 派生当前职位。
5. HUD、帮派树、建筑面板和互动建筑订阅声望，并派生同一等级。
6. 达到建筑阈值时，锁定地块自动切换为 1 级建筑模型。
7. 已解锁建筑的升级仍由 `useCityStore` 执行。

## 错误与边界

- localStorage 不可用时，当前会话仍可运行，只是不持久化。
- 无效或未来的 `lastUpdatedAt` 不产生负收益。
- 单次离线结算最多 8 小时。
- 总声望不超过 1470，等级不超过 50。
- 未知建筑 ID 不被视为已解锁。
- 锁定建筑不能通过直接调用 UI 升级。
- 面板或 modal 点击不得穿透到场景。

## 测试

- 1、8、16、24、32、40、50 对应正确职位。
- 任意中间等级归入正确职位区间。
- 声望 0 对应 1 级，1470 对应 50 级，超额仍为 50。
- 在线收益、离线 8 小时上限、负时间与满级封顶。
- 六座建筑按精确等级和顺序解锁。
- 玩家初始只解锁修车厂。
- 持久 store 同步、reset 与时间结算。
- HUD 显示等级、职位、进度并打开帮派树。
- 帮派树包含 50 节点、七个职位和六个建筑里程碑。
- Escape 与关闭按钮关闭 modal，事件不穿透。
- 锁定建筑面板显示条件且无升级按钮。
- 解锁后建筑面板恢复升级功能。
- 锁定场景使用锁定地块，解锁后使用完整模型。
- 现有建筑升级与城市镜头测试继续通过。

## 验收标准

1. 新玩家刷新后默认等级 1、Prospect，仅拥有修车厂。
2. 声望以每秒 5 点自动增加，等级按 1–50 连续推进。
3. 七个职位在指定阈值正确切换，50 级显示 PRESIDENT。
4. 建筑按“修车厂→废车回收厂→商业街→金属加工厂→加油站→Clubhouse”顺序解锁。
5. 未解锁建筑显示锁定地块和明确条件，不能升级。
6. 解锁后立即显示 1 级完整建筑，并可继续三级升级。
7. 帮派树可查看全部 50 级节点和所有里程碑。
8. 刷新页面可结算本地离线收益，最多 8 小时。
9. 桌面和小屏幕界面均可操作。
10. 全套测试、类型、Lint、格式、构建和 HTTP 冒烟通过。

## 非目标

- 不加入手动领取按钮、付费加速或广告奖励。
- 不加入服务器校时、防作弊或云存档。
- 不加入帮派成员、任务、战斗或分支技能点。
