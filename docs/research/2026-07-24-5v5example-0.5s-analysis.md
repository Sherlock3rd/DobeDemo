## 阶段总结（可复刻的宏观节奏）

1. **五段式主循环**：英雄展示（一次性）→ 章节/关卡地图选关 → 编队部署 → 自动战斗（START→走位→普攻→技能→清场）→ 胜利结算（奖励+进入下一关）→ 回到下一关编队。
2. **单场时长约 8–10 秒**：1-1 从 START（5.5s）到敌亡（14.0s）约 8.5 秒；1-2 从 START（19.5s）到敌亡（26.5s）约 7 秒。适合放置/挂机式短战斗。
3. **逐关递增**：敌方战力 8,100 → 8,108 → 8,650，敌方等级 Lv1 → Lv1 → Lv2；印证「前几关单敌、等级/战力递增」。
4. **参考游戏为半自动**：走位、普攻自动进行；录像中的玩家主动操作是「点击 Enforcer 头像释放技能」（受冷却约束）。DobeDemo 已确认改为技能也自动释放。

## 信息层级（战斗内 HUD 自上而下）

1. **顶部对阵条**：我方头像+总战力（拳头图标）｜关卡号｜敌方总战力+头像。战力是编队页决策的核心指标。
2. **战场层**：敌我单位、各自血条（我方绿、敌方橙红）、命中飘字、枪火/技能特效、死亡布娃娃。
3. **右侧控件**：Auto（自动战斗）、X1（倍速）——教程关锁定。
4. **左上暂停**：`‖`。
5. **底部技能区**：Enforcer 头像（技能键）+ 能量/冷却条 + 冷却倒数字。
6. **结算层**：全屏 VICTORY + 奖励网格 + Next/自动连关。

## 编队 / 战斗 / 胜利节奏要点

- **编队**：拖拽英雄卡到地面虚线阵位；落位后我方总战力实时结算并与敌方战力对比；提供 Quick Deploy（快速布阵）与 Start。阵位是一组固定虚线方框（前排靠上、后排靠下的错列布局）；**本录制全程只部署了 1 名英雄，阵位精确数量与前后排几何不可可靠辨认**，仅能确认存在「多槽、前后分层」的阵型结构。
- **战斗**：START 开场→英雄自动向敌推进→双方远程普攻互扣血→技能冷却就绪后玩家点头像释放多段齐射高伤→清空敌人→判定胜利。技能冷却以整数秒显示；不可用时弹「Can't use the skill right now.」。
- **胜利**：红底 VICTORY→奖励网格（货币/材料/书本/大额经验/英雄碎片/技能书）→Next 进入下一关编队；可解锁「自动连关」。

## 哪些只是「教程手势」（不作为产品规则照搬）

- 红袖手指、黄色鼠标箭头、1-1 节点的绿色手绘圈、「Tap an Enforcer's portrait to use their skill.」「Tap empty space to continue」气泡——均为新手引导演示，不是常驻控件。
- Auto / X1 / Auto-Advance 上的锁形——是「本教程录制账号尚未解锁」的状态，不代表这些功能不存在；恰恰说明产品**存在**自动战斗、倍速、自动连关三项功能。
- 顶部红色 `搜索：步进…(30%)` 文字——录屏/抓帧工具叠层，与游戏无关。

## 哪些是「产品规则」（可提炼进设计）

- 章节→关卡的两级关卡结构；单关有奖励预览与挑战入口。
- 编队：拖拽上阵、总战力对比、快速部署；阵型为前后分层的多槽结构。
- 战斗：自动走位+自动普攻；远程互射；敌我血条；命中飘整数伤害；死亡倒地。
- 技能：由玩家点击 Enforcer 头像主动释放；有冷却（整数秒倒数）；不可用时明确提示；技能为多段/散射高伤。
- 战斗控件：暂停、Auto 自动战斗、倍速（X1/…）、可解锁的自动连关。
- 结算：胜利页 + 奖励；主要产出为经验（EXP 7,500 显著大于其他）。
- 递增曲线：逐关提升敌方等级与战力。

## 与本项目（DobeDemo）的取舍

- **保留并调整**：章节→关卡结构、编队（前排/后排多槽）、总战力对比、暂停/倍速、胜利结算与递增难度；DobeDemo 的走位、普攻和技能均自动执行，保留技能冷却与头像就绪反馈，但不提供手动施法。
- **改造为原创**：把「Enforcer/霰弹枪打手」改为本项目原创中文命名英雄与程序化几何外观；把「金条/袋装物/书本/EXP 药/英雄碎片/技能书」六类奖励简化为**共享英雄经验**为主的产出，落到本项目既有的钱/油/物资经济与新的英雄成长体系，不引入抽卡碎片、技能书等无来源资源。
- **不复刻**：参考游戏的美术、角色、文案、稀有度体系（SR 等）与其付费/道具结构；不做云存档/服务器/PVP。
- **落地为确定性引擎**：参考视频的实时动作在本项目以「纯函数确定性固定步长（100ms tick）战斗引擎 + 独立 R3F 俯视战斗场景」复现，保证可重放与可测试（详见 `docs/superpowers/specs/2026-07-24-campaign-heroes-global-hud-design.md`）。
