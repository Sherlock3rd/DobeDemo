# Beagle 术语表

本文件为项目专有名词的单点引用。文档、规则与代码注释中出现下列术语时，以本表定义为准。

**维护规则**：新增项目专有术语时，必须同步在本文件中补充定义、使用场景，并在相关文档中添加链接。

## Demo

可运行的浏览器端 Web 3D 原型，用于快速验证策划想法。初始 Demo 仅包含示例场景与基础 HUD，不定义具体游戏玩法或美术方向。

## 场景

3D 渲染空间，由相机、灯光、地面和场景对象组成，由 `src/scene/` 下的 React Three Fiber 组件实现。

## HUD

Heads-Up Display，覆盖在 3D 画布之上的 React UI 层，用于显示 Demo 状态（如运行中/已暂停）和操作控件，由 `src/ui/` 实现。

## 游戏状态

跨 3D 场景与 HUD 共享的运行时数据（如暂停标志），由 Zustand store 管理；仅确需跨组件共享的状态才进入 store，玩法规则优先保持为独立 TypeScript 模块。
